# Code of Conduct

## Our Commitment

We are committed to a welcoming, harassment-free, and constructive environment for everyone.

## Expected Behavior

- Be respectful and professional.
- Give actionable and kind feedback.
- Assume positive intent and ask clarifying questions.
- Focus on problem solving and evidence.

## Unacceptable Behavior

- Harassment, discrimination, threats, or personal attacks.
- Trolling, intimidation, or bad-faith disruption.
- Publishing private information without consent.

## Enforcement

Project maintainers may remove or edit comments, issues, or contributions that violate this code and may restrict participation as needed.

## Scope

This Code of Conduct applies to project repositories, issue trackers, discussions, and related communication channels.
